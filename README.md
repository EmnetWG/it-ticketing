IT Ticketing System

An IT support ticketing system built with Node.js, Express, Sequelize (MySQL), and Vanilla JavaScript 
for managing, assigning, and resolving IT-related issues across departments.
User Roles & Permissions

Users: Create and view their own tickets.

Managers: Approve, review, and monitor departmental tickets.

IT Staff: View and manage assigned tickets.

Supervisors: Manage staff, assignments, departments, and oversee overall ticket flow.

Authentication & Security

Secure login using JWT (JSON Web Tokens).

Role-based UI and access control.

Password reset and profile management features.
